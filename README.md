# Utalk-backend

## Setup

1. Configurar variables de entorno en Railway o local:
   - TWILIO_ACCOUNT_SID
   - TWILIO_AUTH_TOKEN
   - TWILIO_WHATSAPP_NUMBER
   - FIREBASE_SERVICE_ACCOUNT_JSON (stringified JSON de la cuenta de servicio)
   - FIREBASE_DATABASE_URL

2. Instalar dependencias:
   ```bash
   npm install
   ```

3. Ejecutar servidor:
   ```bash
   npm start
   ```
